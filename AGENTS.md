# AGENTS.md

Conventions for AI agents working in this repository.

## Responding to review comments

**Always reply to every review comment**, and say what you did:

- If you **addressed** it, reply describing the change and reference the commit
  (e.g. "Narrowed the claim in `<sha>`; it now says …").
- If you **disagree** or are **not making the change**, reply explaining why.

Do not silently fix a comment via a commit without also leaving a reply — the
reply is what tells the reviewer how each point was resolved. A one-line reply
is fine; the point is that no comment is left without a response. This applies
to human and automated (bot) reviewers alike.
